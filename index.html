📖 Example Usage

Fetch streamflow data

fetch("https://unleashedbelial.github.io/colorado-bioregional-api/api/water.json")
  .then(r => r.json())
  .then(d => {
    d.gauges.forEach(g => {
      console.log(g.site_name, g.latest?.value, "ft³/s");
    });
  });

Python example

import urllib.request, json

url = "https://unleashedbelial.github.io/colorado-bioregional-api/api/data.json"
with urllib.request.urlopen(url) as r:
    data = json.loads(r.read())

print("Last updated:", data["last_updated"])
print("Active fires:", data["wildfire"]["fire_count"])
for gauge in data["water"]["gauges"]:
    flow = gauge["latest"]["value"] if gauge["latest"] else "N/A"
    print(f"  {gauge['site_name']}: {flow} ft³/s")
